# PII Classification Prototype

A small data science project exploring how sensitive text can be detected before being sent to an AI system.

The project uses simple regex features and text embeddings to classify prompts into categories such as safe, PII, credentials, confidential business information, source code, and financial data.

This was built as a lightweight prototype to understand the core ideas behind AI data-loss prevention systems: detecting risky inputs, evaluating precision/recall, and analysing where the model fails.
